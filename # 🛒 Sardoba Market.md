# 🛒 Sardoba Market

## Loyiha haqida

Sardoba Market — bu Sardoba tumanidagi barcha do‘konlarni bitta platformaga birlashtiruvchi loyiha.

Platforma orqali:

- Xaridor mahsulotlarni qidiradi va bron qiladi.
- Do‘kon egasi o‘z mahsulotlarini boshqaradi.
- Admin platformani nazorat qiladi.

## Loyiha tarkibi

📱 Mobile — Mijoz ilovasi

⚙️ Backend — Server va API

🛠️ Admin — Admin panel

🌐 Website — Rasmiy web sayt

📂 Shared — Umumiy kodlar

🎨 Assets — Logo va rasmlar

📄 Docs — Hujjatlar

🔧 Scripts — Yordamchi skriptlar
