# 📱 Mobile Application

## Vazifasi

Bu papka Sardoba Market mobil ilovasi uchun mo'ljallangan.

Mobil ilova xaridorlar (mijozlar) foydalanadigan asosiy dastur hisoblanadi.

---

## Ushbu papkada nima bo'ladi?

Flutter loyihasining barcha kodlari shu yerda saqlanadi.

Masalan:

- Bosh sahifa
- Mahsulot qidirish
- Do'konlar ro'yxati
- Mahsulot tafsiloti
- Bron qilish
- Buyurtmalar
- Profil
- Xarita (Location)
- Bildirishnomalar (Notification)
- Sozlamalar

---

## Kim ishlaydi?

Flutter dasturchilar.

Backend yoki Admin dasturchisi bu papkaga kod yozmaydi.

---

## Asosiy vazifalar

- Chiroyli va tez ishlaydigan interfeys yaratish.
- Backend API bilan ishlash.
- Xaridor uchun qulay foydalanish imkonini yaratish.
- Android va iOS qurilmalarini qo'llab-quvvatlash.
